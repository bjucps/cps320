# Requirement Notes

App to do what the Fine Arts email does. Show events and activities in Fine Arts department and others. Program notes, videos, interviews, lectures, live streams, maps. Links to tickets (Showpass). Not selling tickets in the app. Link to donor

Integrate attendance tracking for students who are required to go to a certain number of events (from different app, AttendMe; built in QR code scanner that scans codes at events.) Mu-098 attendance for non-music majors (2 required events). Mu-099 attendance for music majors (5 required events). Ability to export to Excel file for grading reports.

Another app, DigitalConcertHall, can be used as a guide. Or other event apps.

End users are primarily students, some donors and community members as well.

Event programs are at music.bju.edu/music-events. Programs include concerts, recitals, Concert, Drama, Opera series, art exhibitions, shows, lectures, theater productions.

Push notifications by default for canceled events for example.

Possibly use Calendar of Events to populate program list, but have option to add other events.

App opens with menu similar to BJUSafe app. Welcome, Calendar, Donate, etc. 

Update app in real time. Ability to share event through social media (musicatbju, Insta, divcomm, Insta). Ability to pull program templates from database and edit it.
